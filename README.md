# 🚀 Journey AI
> Turn any goal into a structured, AI-powered learning roadmap.
![Journey AI](https://img.shields.io/badge/Built%20with-Claude%20AI-blue) ![Status](https://img.shields.io/badge/Status-Live-green) ![Challenge](https://img.shields.io/badge/DDS-AI%20Challenge%202026-orange)
## 🌟 What is Journey AI?
Journey AI is an AI-powered web application that transforms any learning goal into a personalized, step-by-step roadmap. Instead of getting lost in random YouTube videos and scattered resources — you describe your goal, and AI builds your complete learning path.
**Stop staring at a blank page. Start your journey.**
## ✨ Features
- 🎯 **Goal-to-Roadmap** — Type any goal, get a structured learning path instantly
- 📊 **Progress Dashboard** — Track your journey card by card
- 🔥 **Streak System** — Stay motivated with daily learning streaks
- 🤖 **AI-Powered** — Personalized roadmaps using Claude/OpenAI
- 💾 **Persistent Storage** — Your journeys are saved and always accessible

## 🏃 How to Run Locally
```bash
# Clone the repository
git clone https://github.com/muntherh/journey-ai.git
cd journey-ai
# Install dependencies
npm install
# Set up environment variables
cp .env.example .env
# Add your API key to .env
# Run the app
npm run dev
```
## 🔐 Environment Variables
```env
ANTHROPIC_API_KEY=your_api_key_here
DATABASE_URL=your_postgres_url
```
> ⚠️ Never commit your API keys to the repository.
