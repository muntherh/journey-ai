## 🚀 How to Use Journey AI

**Step 1 — Sign in**
> Open https://journey-ai.replit.app and sign in with Google, Apple, LinkedIn, or Email.

**Step 2 — Type your goal**
> Example goals you can try:
> - `Learn Machine Learning in 3 months`
> - `Become a Full Stack Developer`
> - `Master Data Science from scratch`
> - `Learn Arabic in 6 months`
> - `Start a YouTube channel`

**Step 3 — Get your roadmap**
> In 2–5 seconds, Journey AI generates a complete personalized roadmap with:
> - 📋 Phases and milestones
> - ✅ Step-by-step tasks
> - 🎯 Why each task matters
> - 💪 Practical exercises
> - 📺 YouTube resources with thumbnail previews
> - 🔧 Tools and expert tips
> - ⚠️ Common mistakes to avoid

**Step 4 — Track your progress**
> Use the dashboard to track completed tasks and stay on streak.

## ⚙️ Key Technical Highlights

| Feature | Implementation |
|---|---|
| AI Engine | Claude API (Anthropic) — 16,000 token context |
| Response validation | `finish_reason` check prevents truncated roadmaps |
| Auth | Clerk — Google, Apple, LinkedIn, Email |
| YouTube previews | Computed from video ID — no extra API |
| Deployment | Replit Autoscale |
| Database | PostgreSQL |
